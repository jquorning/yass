Yet Another Static Site (Generator) Documentation

Detailed Information about Program Options

Creating new site project

To create new site project type in directory where file yass is ./yass create [ProjectName] where [ProjectName] is name of your site project. It will be name of directory too. This command creating directory with content:

[ProjectName]
  |-layouts
     |-default.html
  |-output
  |-index.md
  |-site.cfg
  • Directory layouts contains Ada Web Server templates used to generate page.
  • Directory output is directory where generated site will be stored.
  • File index.md is empty Markdown file which will be later main page of site.
  • File site.cfg is configuration file of project.

Building existing project

To build any existing site project, type in directory where file yass is ./yass build [ProjectName] where [ProjectName] is name of your site project. It will convert any not excluded file with extension .md to HTML file, apply layout to it and move with all others, not excluded files to output directory (by default it is output directory in [ProjectName] directory). This command preserve structure of directories and files in your project. Thus if you create directory structure like this:

[ProjectName]
  |-layouts
     |-default.html
  |-output
  |-index.md
  |-site.cfg
  |-images
     |-newimage.png

Your new site will be have that directory structure (inside output directory):

index.html
images
  |-newimage.png

Server mode

To run program in server mode, type in directory where file yass is ./yass server [ProjectName] where [ProjectName] is name of your site project. This will start simple web server at http://localhost:8888/ to which you can point your web browser to preview site. Additionally, program will be monitoring every 5 seconds your site project directory for changes to files. If any file will be changed during server work, it will regenerate whole site.

Limitations

  • At this moment, when you browsing your project, you must enter valid address to file. Going just to http://localhost:8888/ will not works, you must enter http://localhost:8888/index.html
  • Monitoring works only with existing files. If you add new file or delete one, program will not detect it. You must stop, build site manually (if you adding new file) or remove old file manually, then restart server.

Help command

This command gives only short information about all available programs commands. To run it, type in directory where file yass is ./yass help.
